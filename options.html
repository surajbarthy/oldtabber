Tab Aging

Each tab ages on its own (same URL in two tabs = two timers). Optional emoji in the title shows how long since you last used that tab. Use the toolbar popup for quick toggles.

Emoji step timing

Each row is when that emoji appears (idle time since you last focused the tab). Steps must happen in order — if a value would be earlier than the row above, it is adjusted automatically.
